# ADK Agent

This example shows how to quickly setup locally an ADK agent for development purposes. This sample expose an ADK agent on a local web server and it renders a local web UI that allows developers to debug E2E what the agent is doing.

To start the ADK agent locally type the following command:

```sh
uv run adk web --port 8010
```

And open the web UI at: [http://localhost:8010](http://localhost:8010).
More info about [ADK web cli](https://google.github.io/adk-docs/).
